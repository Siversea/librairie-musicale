# librairie-musicale
 Note au client 

 Chaques scripts fonctionnent independament 
 les commandes sont utilisables directement depuis n'importe quel terminal :
 en tapant les commandes : ./Scripts/"et le nom du scripts en question"

 - create_genre : permet de creer un genre et un sous genre 

 - create_artiste : permet de choisir le genre ou le sous genre pour y classer l'artiste à l'interieur de celui ci et d y rentrer "stop"  pour pouvoir arreter la saisie de l'artiste 

 - add_song : Permet de rajouter le son dans la categorie, attribuer à l'artiste, en y rajoutant quelques infos en plus 

 - recherche : Permet de faire une recherche d'une parole et de m'afficher de quel artiste, genre il appartient 

 - export : permet de partager facilement la bibliotheque à quelqu'un d'autre 

 - create_librairie : permet de combiner les 3 premiers scripts, sans interruption pour un gain de temps
